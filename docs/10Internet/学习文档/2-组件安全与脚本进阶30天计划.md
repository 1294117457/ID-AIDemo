- [ ] 加入多线程提高扫描速度
- [ ] 加入命令行参数解析（`argparse`）

### 今日积累
- 子域名枚举是信息收集的关键步骤
- 关键库：`dnspython`（DNS 查询）、`threading`（并发）、`argparse`（CLI 参数）

### 脚本核心逻辑
```python
import socket
import threading

def check_subdomain(subdomain):
    try:
        ip = socket.gethostbyname(subdomain)
        print(f"[+] {subdomain} -> {ip}")
    except socket.gaierror:
        pass

domain = "target.com"
with open("subdomains.txt") as f:
    for line in f:
        sub = f"{line.strip()}.{domain}"
        t = threading.Thread(target=check_subdomain, args=(sub,))
        t.start()
```
