## 〇、测试前准备

### 0.1 环境要求

| 组件 | 要求 |
|------|------|
| Node.js | >= 18 |
| npm | >= 9 |
| Java | JDK 17 |
| MySQL | 8.x，包含 `score_template` 和 `template_rule` 等表 |
| Agent | ID-AIDemo（Express :3001） |
| Backend | idbackend（Spring Boot :8080） |
| Frontend | idfrontend（学生端）/ idfrontend-admin（管理端） |

### 0.2 Agent 依赖安装

```bash
cd LangChain升级文档/project/ID-AIDemo
npm install
```

确保无报错。重点检查以下依赖已正确安装：
- `@langchain/langgraph` ^1.2.8
- `@langchain/mcp-adapters` ^1.1.3
- `@modelcontextprotocol/sdk` ^1.12.0
- `zod` ^3.24.0

### 0.3 环境变量确认

确认 `ID-AIDemo/.env` 包含以下配置（或已通过 SQLite 的 `ai_config` 表配置）：

```
PORT=3001
OPENAI_API_KEY=sk-xxx
OPENAI_BASE_URL=https://xxx
OPENAI_MODEL=xxx
MYSQL_HOST=xxx
MYSQL_PORT=3306
MYSQL_USER=xxx
MYSQL_PASSWORD=xxx
MYSQL_DATABASE=xxx
```

### 0.4 启动顺序

```bash
# 1. 启动 Agent
cd ID-AIDemo && npm start

# 2. 启动 Backend
cd idbackend && mvn spring-boot:run

# 3. 启动前端（学生端 + 管理端）
cd idfrontend && npm run dev
cd idfrontend-admin && npm run dev
```
